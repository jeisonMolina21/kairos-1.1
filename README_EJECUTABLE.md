# 🚀 Kairos - Sistema de Asistencias

## Instalación Automática

### Opción 1: Instalador Automático (Recomendado)
1. Ejecuta `installer.bat`
2. Espera a que se instalen las dependencias automáticamente
3. El ejecutable se creará en `dist/Kairos_Asistencias.exe`

### Opción 2: Manual
1. Instala dependencias: `pip install -r requirements.txt`
2. Construye ejecutable: `python setup.py`
3. Ejecuta: `dist/Kairos_Asistencias.exe`

## Uso del Sistema

1. **Ejecuta** `Kairos_Asistencias.exe`
2. **Selecciona** los archivos Excel de asistencia
3. **Espera** el procesamiento automático
4. **Obtén** el reporte completo en `outputs/`

## Características

- ✅ Ejecutable autónomo - No requiere instalación de Python
- ✅ Instalación automática de dependencias
- ✅ Interfaz gráfica amigable
- ✅ Reportes completos en Excel
- ✅ Conexión automática con API
- ✅ Análisis de puntualidad avanzado

## Estructura de Reportes

1. **Explicación** - Guía completa del sistema
2. **Resumen Semanal** - Vista general por empleado
3. **Diario** - Detalle día a día
4. **Horarios** - Análisis de puntualidad
5. **Horarios Programados** - Horarios asignados
6. **Registros Brutos** - Datos originales

## Soporte

Si encuentras problemas:
1. Ejecuta `installer.bat` nuevamente
2. Verifica que tengas conexión a internet para la instalación automática
3. Asegúrate de que los archivos Excel no estén abiertos al procesar
